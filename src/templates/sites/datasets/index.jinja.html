Sprocket Data Sets

The What:

This website documents the datasets that are provided by the Sprocket platform. They are all provided in the parquet format; which acts similarly to a CSV, but is much smaller.

Parquet is also frequently used with DuckDB for a quick way to start working with the data. DuckDB supports loading parquet files without downloading them, so most examples here will assume that you are using it.

The How:

This is not an exhaustive list of tools that are compatible with the parquet format, but it has a couple options that can help you get started.

The quickest way to get started is with the DuckDB Web Console, which runs an instance of DuckDB in your browser, or with

Command line tools

  • Harlequin A drop-in replacement for the DuckDB CLI.
  • DuckDB CLI The official command line for DuckDB

Browser Based Tools

  • DuckDB Web Console A transient, in-memory database running in your browser, 0 setup required

Desktop Tools

  • Tad A fast, free, cross-platform tabular data viewer application powered by DuckDB.
  • qStudio A free SQL tool specialized for data analysts. It runs on every operating system and allows easy browsing of tables and charting of results.

  • DBeaver See Also

    DBeaver Community is a free cross-platform database tool for developers, database administrators, analysts, and everyone working with data.
